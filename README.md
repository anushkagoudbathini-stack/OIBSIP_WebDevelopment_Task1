# Landing Page Website

## Objective

To design and develop a responsive landing page using HTML and CSS, focusing on clean layout, structure, and basic frontend design principles.

## Steps

1. Planned the layout and structure of the landing page
2. Created the webpage structure using HTML5
3. Applied styling and responsiveness using CSS3
4. Tested the design across different screen sizes

## Technologies Used

* HTML5
* CSS3

## Purpose

This project was developed to strengthen frontend web development skills and demonstrate the ability to build a functional landing page using core web technologies.
